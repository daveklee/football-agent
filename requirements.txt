# ============================================================================
# MCP Server Dependencies
# ============================================================================
# 
# NOTE: The following MCP servers are used but have separate dependencies:
#
# 1. Yahoo Fantasy Football MCP Server
#    - Repository: https://github.com/derekrbreese/fantasy-football-mcp-public
#    - Dependencies: Installed via fantasy-football-mcp-public/requirements.txt
#    - Provides: Yahoo Fantasy Football API access (read-only)
#
# 2. Browser MCP Server
#    - Website: https://browsermcp.io/
#    - Installation: Chrome extension + npm package via npx
#    - Command: npx -y @modelcontextprotocol/server-browser
#    - Provides: Browser automation using your Chrome profile
#
# These are configured in mcp_config.json and handled by the MCP client/ADK.
# No Python packages needed here for MCP servers themselves.

# Optional Dependencies (Not Required for Core Functionality)
